# RP-13: Research rover data sources (traverses, image metadata, panoramas)

Research all publicly available Mars rover data suitable for display on the globe. Rovers of interest: Perseverance (Jezero, 2021–present), Curiosity (Gale, 2012–present), Opportunity (Meridiani, 2004–2019), Spirit (Gusev, 2004–2010), Sojourner (Pathfinder, 1997). For each: (1) Traverse path — does a GeoJSON or CSV of the rover's path exist? Sources: NASA PDS, USGS GIS, JPL; (2) Image metadata — do per-image coordinates exist (lat/lon of where each photo was taken)? Sources: NASA Mars Rover Photos API (api.nasa.gov), PDS image archives; (3) Panoramas — are stitched panoramas publicly available with location metadata? Deliverable: a documented table of what data exists, what format it's in, and what URL/download path to use. This drives the data format decisions for RP-4 (layer registry) before architecture is locked in.
