# RP-9: Layers panel: floating button + layer toggles

Bottom-left floating button (Google Maps layers icon style) that opens a panel. Panel contains: imagery toggle (terraformed vs real Mars, mutually exclusive), and feature toggles (topography lines, exaggerated topography, map labels — checkboxes). Panel styled per RP-6. Extensible: adding a new layer to the registry should surface it in this panel automatically. Depends on RP-4 (layer registry), RP-6 (design system), RP-7 (imagery layer).
